# Parry's Corner Sales Dashboard (Power BI)

## Project Overview

This project focuses on analyzing retail sales data from Parry's Corner to understand business performance, product trends, and profitability. The dashboard provides interactive visualizations that help monitor revenue, profit, and sales patterns for better decision-making.

## Objectives

* Analyze overall sales and profit performance
* Identify top-performing products and categories
* Track monthly sales trends
* Compare revenue and profit
* Support business decisions using data insights

## Tools and Technologies Used

* Power BI
* Microsoft Excel
* DAX (Data Analysis Expressions)
* Data Modeling
* Data Visualization

## Dataset Description

The dataset contains transactional sales data with the following fields:

* Date
* Product Name
* Category
* Quantity Sold
* Selling Price
* Total Revenue
* Total Cost
* Profit

## Key Business Problems Solved

1. Monitored total revenue and profit performance
2. Identified best-selling products and categories
3. Analyzed monthly sales trends
4. Compared revenue versus profit
5. Evaluated category contribution to total revenue
6. Supported performance tracking using KPI metrics

## Dashboard Features

* Interactive sales dashboard
* KPI cards for Revenue, Profit, and Quantity Sold
* Monthly revenue trend visualization
* Category-wise sales analysis
* Product performance insights
* Dynamic filtering using slicers

## Key Insights

* Identified high-performing product categories
* Observed monthly sales growth patterns
* Highlighted profit contribution across products
* Enabled data-driven business decisions
